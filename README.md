## 📌 About the Project

**VOXE MUSIC** is a premium web music portal dedicated to Uzbekistan's top artists. The platform lets users explore albums and watch exclusive music video clips — all in one place, with a clean and modern interface.

---

## ✨ Features

- 🎧 **Albums Section** — Browse song collections from featured artists
- 🎬 **Video Clips** — Watch embedded YouTube music videos directly on the site
- 🎤 **Multiple Artists** — Dedicated pages for each featured musician
- 📱 **Responsive Design** — Works smoothly on all screen sizes
- 🌙 **Dark Theme** — Easy on the eyes, premium feel

## 🚀 Getting Started

```bash
# Clone the repository
git clone https://github.com/abdukarimovyahyo7-web/voxemusic.git

# Navigate into the project folder
cd voxemusic

# Open in browser
open index.html
```
